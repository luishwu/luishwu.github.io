# **App Name**: Profile Pulse

## Core Features:

- Timeline View: Interactive timeline: Display professional experience and projects in an interactive timeline.
- Skills Display: Skill Showcase: Visually appealing skill bars/charts to showcase proficiency.
- Project Presentation: Project Gallery: A gallery to showcase personal projects with images and descriptions.
- AI Feedback: AI-Powered Resume Analysis: tool that provides feedback on resume content and suggests improvements.
- Contact Section: Contact Form: A simple contact form with validation.

## Style Guidelines:

- Primary color: Teal (#008080) for a professional and modern look.
- Secondary color: Light gray (#F0F0F0) for backgrounds and subtle contrasts.
- Accent color: Coral (#FF7F50) for highlights, interactive elements, and calls to action.
- Clean and readable sans-serif fonts for headings and body text.
- Use a consistent set of line icons for skills and sections.
- A clean, single-page layout with clear sections and visual separation.
- Subtle animations and transitions to enhance the user experience.
